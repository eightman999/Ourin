1. 基本方針

  • 語彙互換：UKADOC のキー名・意味を継承。
  • macOS 置換（M‑Diff）：パスは POSIX/URL、座標は左上原点で解釈→内部では AppKit の左下原点に変換。

2. 文字コード/改行

  • 送受信 UTF‑8 標準。CP932/SJIS 受理→内部UTF‑8。
  • 改行は LF（CRLF 受理）。

3. 座標・パス（M‑Diff）

  • *.defaultleft/top 等は 左上原点スクリーン座標として受理。Ourin内部で y' = screenHeight - 1 - y 等に変換。
  • 画像/ログのパスは SHIORIの位置からの相対または file:///POSIX を推奨。
