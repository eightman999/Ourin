# Ourinの拡張機能と動作上の注意

## プラットフォーム焦点

OurinはネイティブmacOSベースウェアです。複数の動作は意図的にSSP互換性より厳密にAppKit/SwiftUI機構を使用しています。

例:

- `NSWindow` APIによるウィンドウ制御とアニメーション
- SwiftUIビューモデルで実装された視覚効果/状態
- ブラウザー、メール、ダイアログのネイティブ設定/開く流れ

## さくらスクリプト実行モデル

パーサーとエグゼキューターが分離:

- `SakuraScriptEngine` がトークン化
- `GhostManager` がトークンを解釈し、再生ユニットをキューイング

これにより、タイピング効果、遅延コマンド、コマンドのタイミング制御をサポートします。

## 拡張moveコマンドハンドリング

`move/moveasync` はオプションスタイル形式をサポート:

- `--X`, `--Y`
- `--time`
- `--base`, `--base-offset`, `--move-offset`
- `--option=ignore-sticky-window`
- `moveasync,cancel`（オプションのスコープ付き）

互換性のため、従来のポジショナル形式も保持されています。

## ウィンドウグループ制御拡張機能

- `set,zorder` はより広いスコープトークン解析をサポート（`current`、`all`、区切り付きリスト）
- `set,sticky-window` グループ化形式をサポート（`--group=...|...` を含む）

これらはmacOSデスクトップでの複数キャラクターウィンドウ制御のための実用的な拡張機能です。

## Dressup/bindハンドリング

`bind` がサポート:

- カテゴリー形式
- 反復タプル形式
- 無効化/バインド解除セマンティック値（`0`、`false`、`off`、`none`、`default`）
