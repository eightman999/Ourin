## メンテナーのためのドキュメント

### 新しいドキュメントの追加
1. `{name}_ja-jp.md` と `{name}_en-us.md` の両方を作成
2. HTMLを生成: `python3 docs/generate_html.py`
3. `docs/html/index.html` を更新して新しいドキュメントを含める
4. `TRANSLATION_MANIFEST.md` にエントリを追加

### 既存ドキュメントの更新
1. マークダウンファイルを編集
2. HTMLを再生成: `python3 docs/generate_html.py`
3. マークダウンとHTML両方の変更をコミット

### 翻訳更新
1. プレースホルダーコンテンツを実際の翻訳に置き換え
2. メタデータを更新（翻訳者、日付、レビュー状況）
3. 「Translation Pending（翻訳保留）」ステータスマーカーを削除
4. HTMLを再生成

- 各フェーズを個別にコミット
- 追跡可能性のための明確なコミットメッセージ
